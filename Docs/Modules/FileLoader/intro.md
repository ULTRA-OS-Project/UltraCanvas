# FileLoader

The unified one-stop loader for images, vector graphics, 3D models, audio, video, text, and spreadsheets. Instead of juggling separate libraries, programmers call one consistent interface that reuses each file plug-in's own save and load logic.
The main benefit of FileLoader is that applications don't need to be updated to support new file formats. For example if there is a new image format only the plugin is loaded for FileLoader and the app is able to load and save the new format. This makes any app future proof. Also new functionality can be provided: when you download an image, audio or video file from a website, you can directly choose another format at save time.
Because format availability depends on the libraries and plugins a build was compiled with, FileLoader also exposes a runtime inventory — `UltraCanvasFileLoader::GetSupportedFormats()` / `UltraCanvasSupportedFormats::GetAll()` — that reports exactly which formats the current build can load and save, and which library or plugin provides each. Loading and saving stay simple and predictable across formats.
