# FileLoader

The unified one-stop loader for images, vector graphics, 3D models, audio, video, text, and spreadsheets. Instead of juggling separate libraries, programmers call one consistent interface that reuses each file plug-in's own save and load logic. 
The main benefit of FileLoader is that applications don't need to be updated to support new file formats. For example if there is a new image format only the plugin is loaded for FileLoader and the app is able to load and save the new format. This makes any app future save. Also new functionaliy can be provided like you want to download an image, audio or video file from a website, you directly can choose another format at save time. 
Loading and saving stay simple and predictable across formats, and a virus check runs automatically every time a file is loaded, keeping applications safe without any extra code.
