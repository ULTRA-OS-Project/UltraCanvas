## Supported File Formats

FileLoader delegates to one plug-in per format family, so the list below grows
automatically as new plug-ins are dropped in — applications never change.

| Format | Extension(s) | Load | Save |
|--------|--------------|:----:|:----:|

| JPEG | `.jpg`, `.jpeg` | ✅ | ✅ |

| AVIF | `.avif` | ✅ | ✅ |
| HEIC / HEIF | `.heic`, `.heif` | ✅ | — |
| GIF | `.gif` | ✅ | ✅ |
| BMP | `.bmp` | ✅ | ✅ |
