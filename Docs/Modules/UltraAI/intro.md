# UltraAI

**UltraAI** is the provider-agnostic AI layer of ULTRA OS — sibling to UltraCanvas (UI) and UltraNet (networking). It gives every app one stable C++ surface for chat, vision, speech, translation, code assist, and image / video / music generation. Behind that surface sit interchangeable providers — cloud services, local models, or in-process mocks — so an app can switch backends without changing a line. Ten capability interfaces each ship with a programmable mock, so the whole module runs with no network and no external model. Adapters are opt-in and self-registering, and every interface exposes a `RawProvider()` escape hatch for full provider-specific access.
