# UltraNet

The unified, secure networking layer for every platform. One API covers HTTP, HTTPS, WebSocket, FTP, FTPS, SFTP, raw TCP and UDP, TLS, and DNS, complete with reusable sessions, proxy support, certificate validation, and async requests for responsive UIs. Additional protocols such as SMTP, IMAP, MQTT, and SSH load as plug-ins without touching the core. Built on libcurl and OpenSSL for proven performance and security.
