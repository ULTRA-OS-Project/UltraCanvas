# VirtualFS

Browse inside archives as if they were ordinary folders. Open ZIP, 7z, TAR, and many more formats without extracting anything first, navigate seamlessly into nested archives, and read files on demand with intelligent caching, streaming for large files, and password support for encrypted content. One consistent, cross-platform interface powers file managers, document viewers, asset loaders, and backup browsers alike.
Same caveat as before: VideoFX, SmartHome, and UltraAI are written from their stated scope, since none has a dedicated master registry in the project knowledge.
