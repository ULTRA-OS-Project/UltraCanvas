# AudioFX

The professional audio engine for UltraCanvas, with 250+ operations across 12 categories. Load, edit, filter, mix, and analyze sound using effects, dynamics, synthesis, spatial processing, pitch and time stretching, and loudness measurement. A streaming architecture handles multi-hour files with minimal memory, and the same API runs identically on Windows, Linux, macOS, and mobile. Built on FFmpeg and libsndfile, with a plug-in system to extend formats and effects.
