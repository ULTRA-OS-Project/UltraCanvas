# FileLoader

A comprehensive, cross-platform file handling system that provides a single, unified API for loading, saving, and converting files across dozens of formats.

## Purpose

FileLoader removes the need to integrate and learn a separate library for every file format. It offers:

- **One API for everything** — load images, audio, video, documents, and 3D models through a single, consistent interface.
- **Effortless format conversion** — convert between compatible formats with a single function call.
- **Built-in security** — automatic malware and virus scanning of loaded content.
- **Cross-platform compatibility** — the same code runs on Windows, Linux, and macOS.

## Supported Formats

| Category   | Formats |
|------------|---------|
| Images     | PNG, JPEG, WebP, AVIF, HEIC, GIF, BMP, TIFF, SVG, PSD, HDR, TGA, RAW |
| Audio      | MP3, FLAC, WAV, OGG, AAC, M4A, OPUS, WMA |
| Video      | MP4, AVI, MKV, WebM, MOV, FLV, WMV |
| Documents  | PDF, DOCX, ODT, RTF, TXT, Markdown |
| 3D Models  | OBJ, FBX, GLTF, STL, 3DS, PLY, COLLADA |
| Archives   | ZIP, RAR, 7Z, TAR, GZ, BZ2 |

## Typical Use Cases

Desktop applications, content management systems, and media processing tools that need uniform handling of diverse file types without taking on a separate dependency per format.
