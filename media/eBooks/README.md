# eBooks

Sample e-books used by the DemoApp eBook Reader page
(`Apps/DemoApp/UltraCanvasEBookExamples.cpp`).

On startup the demo opens this folder and loads
`Game-of-rat-and-dragon.mobi`. Drop that file here to have the reader show it
by default; if the file is missing the demo falls back to a small in-memory
sample book so the page still works.

"The Game of Rat and Dragon" by Cordwainer Smith (1955) is in the public
domain and available from Project Gutenberg
(https://www.gutenberg.org/ebooks/29614) in Kindle/MOBI format.
