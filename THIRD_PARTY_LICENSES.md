## MicroTeX (LaTeX math engine)

- **Used by:** the LaTeX math plugin (`ULTRACANVAS_PLUGIN_LATEX`).
- **Upstream:** https://github.com/NanoMichael/MicroTeX (the `openmath` /
  Uni-math branch).
- **Vendored at:** `UltraCanvas/third_party/microtex/lib`
- **License:** MIT License ŌĆö Copyright (c) 2020 Nano Michael.
- **Full text:** `UltraCanvas/third_party/microtex/LICENSE-MicroTeX.txt`

Local modifications to the vendored copy are limited and clearly marked in the
source with an `[UltraCanvas local patch]` comment:

- `lib/utils/utils.cpp` ŌĆö `defaultLocale()` now probes a list of locale names
  and falls back to the classic "C" locale instead of hard-failing when
  `en_US.UTF-8` is not installed on the host.
- `lib/microtexconfig.h` ŌĆö provided directly (instead of being generated by the
  upstream CMake `configure_file`) for the minimal, path-render build.

The engine is built in **Uni-math** mode with glyphs rendered as vector paths
(`GLYPH_RENDER_TYPE=1`). The legacy Computer Modern / AMS bitmap-era font tree
(`cm*`, `ms*`, `special.ttf`, `res/fonts`, ŌĆ”) is **not** vendored or used.

## Latin Modern Math (math font)

- **Used by:** the LaTeX math plugin as the default OpenType math font, together
  with its pre-generated `.clm2` metrics.
- **Upstream:** GUST e-foundry ŌĆö https://www.gust.org.pl/projects/e-foundry/lm-math
  (also on CTAN: https://ctan.org/pkg/lm-math).
- **Vendored at:** `media/microtex/latinmodern-math.otf` and
  `media/microtex/latinmodern-math.clm2`
- **License:** GUST Font License (GFL), an instance of the LaTeX Project Public
  License (LPPL) ŌĆö freely usable and redistributable, commercial use permitted.
  Copyright 2012ŌĆō2014 B. Jackowski, P. Strzelczyk and P. Pianowski (on behalf of
  TeX Users Groups).
- **Full text:** `media/microtex/LICENSE-LatinModernMath.txt`

The `.clm2` file is a binary metrics + glyph-path table derived from the OTF by
MicroTeX's `prebuilt/otf2clm.py` tool; it is a transformed representation of the
same font and is covered by the same license.

## miniaudio (audio backend)

- **Used by:** the audio subsystem when `ULTRACANVAS_ENABLE_AUDIO` is ON
  (`UltraCanvas/libspecific/Audio/AudioBackendMiniaudio.cpp`). Provides
  cross-platform device I/O, playback/capture streams, WAV encode, and
  WAV/MP3/FLAC decode via its bundled dr_wav/dr_mp3/dr_flac decoders.
- **Upstream:** https://github.com/mackron/miniaudio
- **Vendored at:** `UltraCanvas/libspecific/Audio/miniaudio.h` (single header).
- **License:** dual-licensed as public domain (Unlicense) **or** MIT No
  Attribution (MIT-0), at the user's choice ŌĆö Copyright David Reid. The vendored
  header carries the full license text inline at the top of the file.
- **Note:** the header is vendored unmodified. OGG Vorbis and Opus are not part
  of miniaudio's bundled decoders and are only available when the corresponding
  codec (e.g. stb_vorbis, libvorbis, opusfile) is added to the build; those
  carry their own licenses.
